Вам задано список цілих чисел, і ваша мета - виконати наступні дії:

  • Визначити та згрупувати всі відсортовані послідовності у списку. Відсортована послідовність - це група чисел, розташованих у порядку зростання або спадання.
  • Відсортуйте вхідний список за цими групами, зберігаючи відносний порядок елементів у кожній групі.
  • Повернути відсортований список з розпакованими групами, тобто окремі числа з кожної групи знаходяться у кінцевому результаті, а їх порядок відображає відсортовану послідовність всередині кожної групи.

Наприклад, розглянемо вхідний список [5, 1, 5, 0, 5]:

  • Згрупуйте числа так, щоб отримати [[5, 1], [5, 0], [5]] (виділено три групи).
  • Відсортуйте вхідний список за групами, зберігаючи відносний порядок елементів у кожній групі, щоб отримати [[5], [5, 0], [5, 1]].
  • Розпакуйте групи, щоб отримати остаточний відсортований список: [5, 5, 0, 5, 1].

Важливі моменти:

  • При визначенні відсортованих груп сусідні рівні числа вважаються частиною однієї групи і зберігають раніше визначений порядок.
  • Щоб визначити порядок сортування для кожної групи, шукайте перше відмінне число за межами початкових рівних чисел. Порядок сортування для кожної групи визначається не один раз для всього списку.
  • При створенні груп, будьте жадібні зліва, беручи якомога більше чисел, щоб створити чітку впорядковану послідовність для кожної групи.
