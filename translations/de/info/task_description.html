Sie erhalten eine Liste ganzer Zahlen und Ihr Ziel ist es, die folgenden Schritte auszuführen:

  1. Identifizieren und gruppieren Sie alle sortierten Sequenzen innerhalb der Liste. Eine sortierte Folge ist eine Gruppe von Zahlen, die entweder in aufsteigender oder absteigender Reihenfolge angeordnet sind.
  2. Sortieren Sie die Eingabeliste nach diesen Gruppen und behalten Sie dabei die relative Reihenfolge der Elemente innerhalb jeder Gruppe bei.
  3. Geben Sie die sortierte Liste zurück, wobei die Gruppen entpackt sind, d.h. die einzelnen Zahlen aus jeder Gruppe sind im Endergebnis enthalten und ihre Reihenfolge entspricht der sortierten Reihenfolge innerhalb jeder Gruppe.

Nehmen wir zum Beispiel die Eingabeliste [5, 1, 5, 0, 5]:

  • Gruppieren Sie die Zahlen, um [[5, 1], [5, 0], [5]] zu erhalten (drei Gruppen werden identifiziert).
  • Sortieren Sie jede Gruppe unabhängig voneinander und behalten Sie dabei die relative Reihenfolge der Elemente innerhalb jeder Gruppe bei, um [[5], [5, 0], [5, 1]] zu erhalten.
  • Entpacken Sie die Gruppen, um die endgültige sortierte Liste zu erhalten: [5, 5, 0, 5, 1].

Wichtige Punkte:

  • Bei der Identifizierung von sortierten Gruppen werden benachbarte gleiche Zahlen als Teil der gleichen Gruppe betrachtet.
  • Um die Sortierreihenfolge für jede Gruppe zu bestimmen, suchen Sie nach der ersten unterschiedlichen Zahl nach den anfänglichen gleichen Zahlen. Die Sortierreihenfolge für jede Gruppe wird nicht einmal für die gesamte Liste festgelegt.
  • Seien Sie bei der Bildung der Gruppen von links gierig und nehmen Sie so viele Zahlen wie möglich, um eine eindeutige Reihenfolge für jede Gruppe zu erstellen.
