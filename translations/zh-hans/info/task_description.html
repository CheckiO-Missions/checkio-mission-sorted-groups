给你一个整数列表，你的目标是完成以下步骤：

  • 找出列表中的所有排序序列，并将它们组合在一起。排序序列是按升序或降序排列的一组数字。
  • 根据这些分组对输入列表进行排序，同时保留每个分组内元素的相对顺序。
  • 在返回排序后的列表时，将各组解压缩，这意味着每个组中的单个数字都会出现在最终结果中，它们的顺序反映了每个组内的排序序列。

例如，考虑输入列表[5, 1, 5, 0, 5] ：

  • 将数字分组，得到[[5, 1], [5, 0], [5]] （确定了三个组）。
  • 按组对输入列表排序，同时保持各组内元素的相对顺序，得到[[5], [5, 0], [5, 1]] 。
  • 解压缩分组，得到最终的排序列表：[5, 5, 0, 5, 1].

要点

  • 在识别排序组时，相邻的相等数字被视为同一组的一部分，并保持之前检测到的顺序。
  • 要确定每组的排序顺序，应查找初始相等数字之外的第一个不同数字。每个组的排序顺序不会一次确定整个列表的排序顺序。
  • 在分组时，要从左侧开始贪心地尽可能多取数字，为每组创建一个不同的有序序列。
